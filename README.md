 #### 目录结构

安装软件过程中产生的包，图标，截图等被储存到`/tmp/deepin-community-store/`中。

配置文件被储存到`~/.config/deepin-community-store/`中。

线路文件：新版的线路文件被放置于源服务器中，可随时刷新更新源列表
